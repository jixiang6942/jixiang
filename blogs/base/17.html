# 三，深入理解pixel、DPR、PPI

# 1，pixel(像素)

# 2，像素的分类

  • 物理像素：设备屏幕横向或纵向有多少个物理发光点。
  • 逻辑像素：操作系统可以控制多少个物理发光点表示一个像素。

# 3，DPR、PPI

高清屏幕下，水平方向，一个逻辑像素需要两个物理像素来显示。因此物理像素和逻辑像素的比值，我们称之为DPR。

总结：书写的元素在页面中默认实在980px宽度的布局视口中。如果使用移动端打开网页，在没有进行任何配置的情况下，为了在移动端能够完整展示980px宽度的布局视口，会对布局视口进行压缩，与之对应的是，布局视口中的元素也会被压缩。因此我们需要进行设置width='device-width'，让布局视口等于视觉视口。
